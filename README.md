# 🚗 Car Price Prediction Web Application

A Machine Learning-powered web application that predicts the price of a used car based on various features such as company, car model, manufacturing year, fuel type, and kilometers driven.

## 📌 Project Overview

This project uses a Linear Regression model to estimate the market price of a used car. The model is trained on a cleaned dataset and deployed through a Flask web application, providing an interactive user interface for real-time predictions.

## ✨ Features

* Predicts used car prices instantly
* User-friendly web interface
* Machine Learning-based predictions
* Data preprocessing and cleaning
* Flask-powered backend
* Responsive frontend using HTML and CSS

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Flask
* HTML
* CSS

## 📂 Dataset

The dataset contains information about used cars, including:

* Company
* Car Name
* Manufacturing Year
* Fuel Type
* Kilometers Driven
* Price

Data cleaning and preprocessing were performed before training the model.

## 🤖 Machine Learning Model

* Algorithm: Linear Regression
* Data Splitting: Train-Test Split
* Feature Encoding: One-Hot Encoding
* Evaluation Metric: R² Score

## 🚀 Installation & Usage

1. Clone the repository:

```bash
git clone https://github.com/your-username/car-price-prediction.git
```

2. Navigate to the project directory:

```bash
cd car-price-prediction
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the Flask application:

```bash
python app.py
```

5. Open your browser and visit:

```bash
http://127.0.0.1:5000
```

## 📸 Application Workflow

1. Select Car Company
2. Enter Car Model
3. Choose Manufacturing Year
4. Select Fuel Type
5. Enter Kilometers Driven
6. Click "Predict Price"
7. View the predicted car price instantly
